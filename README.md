# 健康管理系统 (Health Management System)

## 技术栈

- 后端: FastAPI + SQLAlchemy + Pydantic
- 前端: React 18 + Vite + Ant Design 5
- 数据库: MySQL 5.7
- 包管理: pnpm workspaces

## 开发

```bash
# 启动所有服务
docker-compose up -d

# 后端: http://localhost:8000
# 前端: http://localhost:3001
# API 文档: http://localhost:8000/docs
```
